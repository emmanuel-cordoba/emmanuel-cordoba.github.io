Command Line Guide

Intro to Command Line

In the table below you can find some of the basic commands to use in a Terminal


Command Description
pwd It displays the file path of the current directory
mkdir It is used to make a new directory in the filesystem according to its argument
ls It is used to list the contents of a directory. ls -a lists all contents, including hidden files and directories. ls -l, lists all contents in long format. ls -t, list all content by the time they were last modified
cd It is used to move throughout the filesystem of a computer
touch It created a new file in the current working directory
cp To copy files or directories. cp - source - destination
mv It moves a file into a directory. mv - source file - destination directory
rm It deletes files and directories. rm -r will delete a directory and all of its content
clean It cleans the terminal
